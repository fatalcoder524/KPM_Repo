# Host File Redirect
Host File Redirect is KPM uses similar or same concept of SusFS feature of susfs_open_redirect to redirect all syscalls from /system/etc/hosts to /data/adb/hosts developed by @sekaiacg

Follow him and support him on GitHub here: https://github.com/sekaiacg

KPMs are normally released in TG Group: https://t.me/apatch_discuss

All Rights belong to the original developer! I'm just keeping this for my personal use or quick indexing and sharing. 

## Note:
- Use it along with [Bindhosts](https://github.com/bindhosts/bindhosts) module.
- Use bindhosts mode 3.
